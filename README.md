# 💰 Target Financeiro (React Native - Expo)

Aplicativo mobile para gerenciamento de metas financeiras, desenvolvido com React Native utilizando Expo.

---

## 📱 Sobre o projeto

O **Target Financeiro** é um app simples e funcional que permite ao usuário:

* Criar metas financeiras
* Definir valores alvo
* Acompanhar progresso
* Registrar entradas (guardar dinheiro)
* Registrar saídas (resgatar dinheiro)
* Editar metas
* Excluir metas

Tudo isso com uma interface leve e intuitiva.

---

## 🚀 Tecnologias utilizadas

* React Native
* Expo
* JavaScript (Hooks)
* Componentes nativos (View, Text, FlatList, etc.)

---

## ▶️ Como executar o projeto

### 🔧 Pré-requisitos

Antes de começar, você precisa ter instalado:

* Node.js
* Expo CLI (ou usar via npx)

---

### 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

Acesse a pasta do projeto:

```bash
cd SEU-REPOSITORIO
```

Instale as dependências:

```bash
npm install
```

---

### ▶️ Executando o app

Inicie o projeto com:

```bash
npx expo start
```

Depois disso, você pode:

* Abrir no celular com o app **Expo Go**
* Rodar no emulador Android/iOS
* Rodar no navegador

---

## 📲 Executar via Snack (sem instalar nada)

Você também pode rodar diretamente online usando o:

👉 Expo Snack

Basta colar o código no editor e executar.
